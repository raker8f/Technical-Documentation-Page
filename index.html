Introduction

C# is a modern, object-oriented programming language developed by Microsoft as part of the .NET platform. It is widely used for building desktop applications, web services, games (using Unity), and more.

C# emphasizes readability, type safety, and productivity. Its syntax is similar to Java and C++, making it accessible to developers with experience in those languages.


    using System;

    class Program {
        static void Main() {
            Console.WriteLine("Hello, C#!");
        }
    }
Variables

Variables are used to store data in a C# program. Each variable has a specific type that determines what kind of data it can hold, such as int, string, or bool.

You must declare a variable before using it, and you can optionally assign it a value.


    int age = 25;
    string name = "Alice";
    bool isStudent = true;
Data Types

C# is a statically-typed language, which means you must specify the type of a variable when declaring it. Common data types include:

  • int for integers
  • double for floating-point numbers
  • bool for boolean values
  • char for single characters
  • string for text

You can also define your own types using class or struct.


    double price = 19.99;
    char grade = 'A';
    string message = "Welcome to C#!";
Control Flow

Control flow statements determine the execution path of your program. C# includes:

  • Conditional statements like if, else if, else
  • Looping structures like for, while, foreach

These structures help you perform tasks based on conditions or repeat tasks multiple times.


    int number = 10;

    if (number > 0) {
        Console.WriteLine("Positive number");
    } else {
        Console.WriteLine("Non-positive number");
    }
      

    for (int i = 0; i < 5; i++) {
        Console.WriteLine("Count: " + i);
    }
Methods and Classes

C# uses classes to define custom types and methods to define reusable blocks of code. Methods are typically placed inside classes.

A method can accept parameters and return a value. Classes can hold data (fields) and behavior (methods).


      class Calculator {
          public int Add(int a, int b) {
              return a + b;
          }
      }

      class Program {
          static void Main() {
              Calculator calc = new Calculator();
              Console.WriteLine(calc.Add(5, 3)); // Output: 8
          }
      }
